# 地圖資料說明

## 台灣縣市地圖 GeoJSON

如果地圖無法從線上載入，您可以：

1. **下載台灣縣市 GeoJSON 檔案**，放置在此目錄並命名為 `taiwan.json`

2. **推薦的資料來源**：
   - 政府資料開放平台：https://data.gov.tw/dataset/7442
   - GitHub: https://github.com/kiang/pharmacies/tree/master/json
   - GitHub: https://github.com/g0v/twgeojson

3. **檔案格式**：GeoJSON (.json)

4. **檔案命名**：`taiwan.json`

## 自動下載（可選）

系統會自動嘗試從多個線上資源載入地圖，如果都無法載入，會使用簡化的矩形地圖作為備用方案。
