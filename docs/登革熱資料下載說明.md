# 台灣登革熱病例開放資料下載說明

## 資料來源

### 1. 政府資料開放平台 (data.gov.tw)
- **網址**: https://data.gov.tw/datasets/search?qs=登革熱
- **說明**: 可搜尋和下載登革熱相關的開放資料集
- **資料格式**: CSV, JSON, XLS, XLSX

### 2. 疾病管制署傳染病統計
- **網址**: https://nidss.cdc.gov.tw/nndss/DiseaseMap?id=061
- **說明**: 提供登革熱的流行趨勢圖、統計報表和地理分布
- **更新頻率**: 每日更新

### 3. 傳染病統計資料查詢系統
- **網址**: https://nidss.cdc.gov.tw/nndss/
- **說明**: 可查詢和下載多種傳染病的統計資料，包括登革熱
- **功能**: 提供 Excel 報表下載、地理分布視覺化

## 使用方法

### 方法一：使用 Python 腳本自動下載

```bash
# 互動模式（會詢問是否下載）
python src/download_dengue_data.py

# 自動模式（直接搜尋和下載）
python src/download_dengue_data.py --auto
```

### 方法二：手動下載

1. 前往 [政府資料開放平台](https://data.gov.tw/datasets/search?qs=登革熱)
2. 搜尋「登革熱」相關資料集
3. 選擇需要的資料集並下載
4. 將下載的檔案放置到 `data/raw/` 目錄

### 方法三：從疾病管制署網站下載

1. 前往 [疾病管制署傳染病統計](https://nidss.cdc.gov.tw/nndss/DiseaseMap?id=061)
2. 選擇需要的時間範圍和地區
3. 下載 Excel 報表
4. 將檔案轉換為 CSV 格式（如需要）並放置到 `data/raw/` 目錄

## 資料欄位說明

根據現有的 `Dengue_Daily.csv` 資料，主要欄位包括：

- **發病日**: 病例發病日期
- **個案研判日**: 病例確診日期
- **通報日**: 病例通報日期
- **性別**: 病例性別
- **年齡層**: 病例年齡層
- **居住縣市**: 病例居住縣市
- **居住鄉鎮**: 病例居住鄉鎮
- **居住村里**: 病例居住村里
- **感染縣市**: 感染地點縣市
- **感染鄉鎮**: 感染地點鄉鎮
- **是否境外移入**: 是否為境外移入病例
- **感染國家**: 境外移入的感染國家
- **確定病例數**: 病例數（通常為1）
- **血清型**: 登革熱血清型（第一型、第二型、第三型、第四型）

## 注意事項

1. **資料更新**: 開放資料通常每日更新，建議定期下載最新資料
2. **資料格式**: 不同來源的資料格式可能不同，下載後可能需要進行資料清理
3. **編碼問題**: 如果遇到中文亂碼，請使用 UTF-8 或 UTF-8-sig 編碼讀取
4. **隱私保護**: 資料已去識別化，但仍需注意使用規範
